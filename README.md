# Frontend Mentor - News homepage solution

This is my solution to the [News homepage challenge on Frontend Mentor](https://www.frontendmentor.io/challenges/news-homepage-H6SWTa1MFl). 

## Overview

### The challenge

I successfully achieved the following objectives:
- Optimized layout for both desktop and mobile using CSS Grid and Flexbox.
- Interactive hover and focus states for all navigable elements.
- Functional mobile sidebar menu with a darkened background overlay.

## My process

### Built with

- Semantic HTML5 markup (proper use of `<main>`, `<section>`, `<aside>`, `<article>`).
- CSS custom properties (Variables).
- Flexbox for internal component alignment.
- CSS Grid for the main page layout.
- Vanilla JavaScript for the sidebar menu logic.
- `<picture>` tag for responsive image handling.

### What I learned

In this project, I reinforced concepts that I previously handled in a more manual or less structured way, now applying a professional approach:

*   **Semantic Structure**: I learned to organize content based on its actual meaning in the document, not just its visual appearance.
*   **Advanced CSS Grid**: Using `grid-column: span` to create areas that occupy multiple columns and how to reset them within Media Queries.
*   **Positioning**: Implementing a Sidebar with `position: fixed` and managing visual layers with `z-index` for the overlay.
*   **DOM Manipulation**: JavaScript logic to open and close menus by changing styles in real-time.
*   **Best Practices**: Understanding the crucial differences between using `<button>` vs. `<a>`, and `<picture>` vs. `<img>` for performance and accessibility.
